# Imagens do Hero

Coloque aqui a imagem de fundo da seção principal (Hero).

**Arquivo esperado pelo CSS:** `tubulacao-industrial.jpg`
(referenciado em `assets/css/style.css` na classe `.hero`)

Recomendações:
- Formato: `.webp` (preferencial) ou `.jpg` otimizado.
- Resolução: 1920x1080 ou superior.
- Peso: idealmente < 300 KB (use compressão) para nota alta no Lighthouse.
- Tema: tubulação industrial / planta industrial.
