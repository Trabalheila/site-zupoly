# Imagens de Projetos / Portfólio

Coloque aqui as imagens dos projetos exibidos nas seções **Quem Somos** e **Portfólio**.

**Arquivo usado em "Quem Somos":** `projeto-3d-tubulacao.jpg`

**Portfólio:** adicione as imagens e atualize os itens da galeria em `index.html`
(cada `.gallery-item` possui `data-full="assets/img/projetos/SEU-ARQUIVO.jpg"`
para abrir no lightbox e `data-category` para os filtros).

Categorias de filtro disponíveis: `tubulacao`, `3d`, `2d`, `layout`, `industrial`.

Recomendações:
- Formato `.webp` ou `.jpg` otimizado.
- Proporção sugerida: 4:3.
- Nomeie de forma descritiva (ex.: `isometrico-linha-vapor.webp`).
